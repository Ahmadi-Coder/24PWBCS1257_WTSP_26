# Lab05 Layouts

This project demonstrates different CSS layout techniques and responsive design practices.

## 📂 Project Structure
- **index.html** — Central hub linking to all layouts and exercises.
- **styles/** — Contains CSS files for base styles, layout techniques, media queries, responsive utilities, and preferences.
- **layouts/** — Demo pages for different layout patterns.
- **exercises/** — Practice exercises for display, float, position, flexbox, grid, and media queries.
- **README.md** — Documentation for the project.

## 📝 Tasks Implemented
- **Task 16: Common Breakpoint Implementation**  
  Added responsive media queries for mobile, tablet, small desktop, large desktop, and extra large screens.

- **Task 17: Responsive Utility Classes**  
  Created utility classes for show/hide elements, responsive spacing, text alignment, and flexbox utilities.

- **Task 18: Preference Media Queries**  
  Implemented `prefers-color-scheme` (dark/light mode), `prefers-reduced-motion`, `prefers-contrast`, and `print` styles. Added a manual dark/light toggle button.

## 🚀 How to Use
1. Open `index.html` in a browser to navigate to all layouts and exercises.
2. Resize the browser window to test responsive breakpoints.
3. Switch system preferences (dark mode, reduced motion, high contrast) to see adaptive styles.
4. Use the **Toggle Theme** button in `mobile-landing.html` to manually switch between dark and light mode.
5. Try Print Preview to check print styles.
